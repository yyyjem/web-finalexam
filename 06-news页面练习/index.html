JavaScript 核心语法学习一

变量与数据类型

在 JavaScript 中，我们可以使用 let、const 声明变量。const 用于声明常量，一旦赋值就不能修改；let 用于声明可变变量。

const a = 100; // 数值类型
const apiKey = 'SECRET_KEY'; // 字符串类型

分支语句

if...else 语句用于根据条件执行不同的代码块。

const currentTime = '12:20';
if (currentTime === '12:20') {
console.log('冲啊！');
} else {
console.log('假装学习……');
}

循环语句

JavaScript 提供了多种循环语句，如 for 和 while 循环，用于重复执行代码块。

// for 循环
for (let i = 1; i <= 10; i++) {
console.log(i);
}

// while 循环
let hug = 60;
while (hug > 0) {
console.log('继续干饭……');
hug -= 10;
}

函数

函数是 JavaScript 中的基本组成部分，用于封装可复用的代码块。

const intro = () => {
console.log('你好，我是一个脚本！');
};
intro();
