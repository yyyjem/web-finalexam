CSS 核心概念：背景、盒模型、布局、动画

背景

在 CSS 中，背景（Background）属性可以用来设置元素的背景颜色、背景图片、平铺方式、位置等。例如页面的整体背景色能营造不同的视觉氛围，而元素的背景图片可以增强页面的视觉效果。

盒模型

盒模型（Box Model）是 CSS 布局的基础，每个元素都被看作一个矩形盒子，由内容（content）、内边距（padding）、边框（border）和外边距（margin）组成。

动画

CSS 动画允许元素在一定时间内从一种样式逐渐变化为另一种样式。动画通常需要定义起始状态、结束状态和持续时间。
