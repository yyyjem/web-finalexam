正常信息流

在正常信息流中，元素会按照 HTML 文档的顺序依次排列，每个元素占据自己的空间。

这是一个普通段落，它会在文档流中正常显示。

此段落展示正常文档流中的元素样式。

所有元素都遵循从上到下、从左到右的布局规则。

正常信息流是网页布局的基础模式。

相对定位

相对定位（position: relative）会相对于元素在正常文档流中的位置进行偏移。

元素偏移后，原位置仍会被保留。

这个段落使用了相对定位，向下偏移了 150px，但原位置空间依旧保留。

相对定位常用于微调元素的位置，而不影响其他元素的布局。

下面是相对定位的 CSS 代码示例：

.relative { position: relative; top:150px;}

绝对定位

绝对定位（position: absolute）会使元素脱离正常文档流，相对于最近的非静态定位祖先元素进行定位。

如果没有找到非静态定位的祖先元素，则相对于 <html> 元素定位。

这个段落使用了绝对定位，向下偏移 150px，并且不会保留原位置空间。

绝对定位常用于创建弹出窗口、悬浮菜单等特殊布局。

下面是绝对定位的 CSS 代码示例：

.absolute { position: absolute; top:150px;}
